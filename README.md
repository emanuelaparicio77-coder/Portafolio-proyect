# Portafolio-proyect

## SMASH & FUEGO — Food Truck Landing Page

Sitio web estático para un food truck ficticio, con estética urbana/callejera (negro, naranja fuego, amarillo mostaza). Construido como pieza de portafolio.

**Demo en vivo:** https://foodtruck-web-omega.vercel.app

### Stack
- HTML + CSS + JavaScript vanilla (sin frameworks ni dependencias)
- Un solo archivo `index.html`
- Deploy en Vercel

### Features
- Countdown en vivo hasta la hora de apertura/cierre
- Menú filtrable por categoría (sin recargar la página)
- Carrusel de reseñas automático
- Formulario de pedido anticipado que genera un mensaje de WhatsApp
- Tabla de horarios con el día actual resaltado dinámicamente
- Animaciones activadas por scroll (Intersection Observer)
